The Worldguard Beginnings - Gra komputerowa, której głównym celem było sprawdzenie moich nabytych umiejętności programistycznych.

Wersja, która jest tutaj dostępna, nie posiada chwilowo muzyki i pozwala na zagranie na tylko kilku gotowych poziomach.

Można skorzystać z pliku .exe, aby w nią zagrać.

W folderze "hct" znajdują się pliki z moimi własnymi implementacjami kilku struktur danych, algorytmów i innych przydatnych narzędzi programistycznych.
